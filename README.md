# jojonomic_test_leonard

## Requirements
  - Golang 
  - Go Module
  - Postgresql Database

## Installing
  - Install Dependecies
     ```console in each microservices
     $ go mod tidy
     ```
  - Fill your local .env configuration

## Run Program
  - Run Project
     ```console in each microservices
     $ go run cmd/main.go
     ```
  - Run Docker
       ```console
    docker-compose -f ./misc/docker-compose.yaml up -d
     ```
